سؤال: معلومات عن مكتبة ReactJS وكيف تتعلمها؟؟
جواب: 
ال React هيي مكتبة جافا سكربت مفتوحة المصدر تم تطويرها بواسطة فيسبوك، وهي تستخدم لبناء واجهات مستخدم .
لحتى نصير مطورين رايكت لازم يكون عنا معرفة ببعض الأمور بالأول :
HTML (HyperText Markup Language)
ال HTML هي لغة وسمية تُستخدم لبناء هياكل صفحات الويب ,و يستخدم وسوم (tags) لتحديد العناصر .
كيف منقدر نتعلمها ؟
اسهل شي ممكن الواحد يتعلمو اي بلاي ليست على اليوتيوب او موقع W3School .
CSS (Cascading Style Sheets):
ال CSS هي لغة تنسيق تُستخدم لجعل صفحات الويب تبدو جميلة ومنظمة , و تستخدم ال CSS لتحديد تصميم الصفحة وتنسيق العناصر التي تم إنشاؤها باستخدام HTML , كما يمكنك استخدام CSS لتغيير الألوان، والخطوط، والهوامش، والتصاعد والتصغير، والتخطيط، وغيرها من الخصائص لعناصر HTML.
متل ماحكينا عن ال HTML نفس مصادر التعلم .
في كمان frameworks لل CSS متل Bootstrap و Tailwind انا من جهتي بفضل ال Tailwind وبنصح جداً بتعلمها وفي كتير عنها بالـ يوتوب أو الدكيومنتيشن الرسمي الها .
منجي للجد هلق
:
لازم نتعلم JavaScript لأن ال React بيعتمد بشكل أساسي عليها , وهيي لغة كتير حلوة وجميلة وسهلة وفيها كلشي , صح كتير كبيرة بس مش عاوزين كلشي فيها بس من المهم جدا نتشعب ونتعلم وناخد وقتنا ونغوص باللغة شوي ونعطيها حقها لان صدقاً قادرين نعمل فيها اي شي بالعالم .
ال JavaScript الها كتير مصادر تعلم وكتير فيديوهات يوتيوب وفي كمان نفس الموقع W3School شارح كلشي باللغة .
المرحلة التاية هيي تعلم React ,بس من الأخبار الحلوة انو ال React عندا documentation من أجمل مايكون وكتير متعوب عليها :
https://react.dev/
